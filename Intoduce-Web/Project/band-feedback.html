MY PORTFOLIO

개인 프로젝트 [My Music]

합주 날짜별 밴드 및 파트별 피드백을 입력 및 관리하고 확인할 수 있는 웹 프로젝트

주요 기능

  • 합주 날짜 별 피드백 생성
  • 각 파트 및 밴드 피드백 입력
  • 각 파트 및 밴드 피드백 확인
  • 피드백 삭제

기간

2026.01.22 ~ 2026.02.01

사용 기술

  • Web : HTML, CSS, JavaScript
  • Server : Java, Spring Boot, Spring Boot Web, Spring Boot JPA, Lombok, Gradle
  • DB : MySQL

상세 내용

BandFeedback 프로젝트는 밴드 합주 활동에서 각 파트 및 전체 밴드에 대한 피드백을 효율적으로 기록, 조회, 관리할 수 있도록 웹 기반 시스템을 구축한 프로젝트입니다. 서버는 Spring Boot를 기반으로 REST API를 제공하며, JPA를 활용하여 BandFeedback과 MemberFeedback 데이터를 MySQL 데이터베이스에 저장하고 관리합니다

주요 기능으로는 합주 날짜별 전체 밴드 피드백 등록, 조회, 삭제와 멤버별 피드백 등록, 조회, 삭제, 저장된 날짜별 피드백 목록 조회, 데이터 무결성을 위한 날짜별 존재 여부 확인 기능 등이 있습니다. 각 엔티티는 Lombok을 활용해 코드 간결성을 유지하고, JPA 쿼리를 통해 복잡한 데이터 조회 및 삭제 작업을 효율적으로 처리했습니다.

해당 작업물의 목표는 최근 빠르게 변화하는 트렌드를 사람들이 쉽고 빠르게 접할 수 있도록 하기 위한 서비스 제공입니다. 단순히 쉬운 접근만이 아니라 트렌드를 쉽게 이해하고 그 추이를 가늠할 수 있는 시각적 자료를 제공하여 이용자가 잘 받아들이고 이를 활용할 수 있는 것이 목표였습니다.

또한 REST API 설계를 통해 프론트엔드와의 연동을 단순화하여 사용자가 웹에서 직관적으로 피드백을 관리할 수 있도록 구현했습니다. 이 프로젝트를 통해 서버-DB 연동, REST API 설계, 데이터 관리 및 협업 구조 설계 경험을 쌓았으며, 실제 밴드 활동 환경에 맞춘 실무형 프로젝트를 완성할 수 있었습니다.
