MY PORTFOLIO

개인 프로젝트 [My Music]

앨범별 곡 재생과 DB 연동으로 곡 정보를 웹에서 확인할 수 있는 프로젝트

주요 기능

  • 곡 재생
  • 곡 정보 확인
  • 앨범 선택 가능

기간

2026.01.22 ~ 2026.02.01

사용 기술

  • Web : HTML, CSS, JavaScript
  • Server : Java, Spring Boot, Spring Boot Web, Spring Boot JPA, Lombok, Gradle
  • DB : MySQL

상세 내용

MyMusic 프로젝트는 사용자가 앨범별 곡을 웹에서 재생하고, 서버를 통해 데이터베이스에 저장된 곡 정보를 불러와 보여주는 웹 기반 음악 재생 서비스입니다. 서버는 Spring Boot를 기반으로 구축하였고, JPA를 활용하여 곡 정보를 MySQL 데이터베이스에 관리하였습니다.

주요 기능으로는 앨범별 곡 조회, MP3 파일 재생, 곡 정보 조회, 서버와 데이터베이스 연동을 통한 곡 데이터 관리 등이 있으며, REST API를 활용하여 프론트엔드와의 원활한 통신을 구현했습니다. 또한 Lombok을 활용하여 엔티티 클래스의 getter/setter 및 생성자 코드를 간결하게 유지하고, Spring Boot의 의존성 주입과 컨트롤러 구조를 통해 코드 재사용성과 유지보수성을 높였습니다.

해당 작업물의 목표는 최근 빠르게 변화하는 트렌드를 사람들이 쉽고 빠르게 접할 수 있도록 하기 위한 서비스 제공입니다. 단순히 쉬운 접근만이 아니라 트렌드를 쉽게 이해하고 그 추이를 가늠할 수 있는 시각적 자료를 제공하여 이용자가 잘 받아들이고 이를 활용할 수 있는 것이 목표였습니다.

이 프로젝트를 통해 서버-DB 연동, REST API 설계, 음악 재생 기능 구현, 프론트엔드-백엔드 연동 경험을 쌓았으며, 실제 사용자가 음악을 탐색하고 재생할 수 있는 서비스 환경을 구현함으로써 실무형 웹 서비스 개발 경험을 강화할 수 있었습니다.
